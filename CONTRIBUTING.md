# Contributing to Nigam

## Most Wanted

### 1. Anu Malik Roast Lines
The `/ask-anu-malik` command needs an ever-expanding arsenal. Submit PRs with:
- New roast templates for specific code patterns
- Cultural references (Bollywood, cricket, food)
- Domain-specific burns (ML, blockchain, frontend, DevOps)

Rules: Technically accurate. Funny. Never cruel about the person, only the code.

### 2. Domain Judges
New judges for specific domains, named after Indian music legends:
- **Judge Vishal** (Vishal Dadlani) — Security audits with punk rock energy
- **Judge Shreya** (Shreya Ghoshal) — Frontend/UX with classical precision  
- **Judge AR Rahman** — ML/Data with Oscar-winning standards
- **Judge Badshah** — Quick-and-dirty rapid prototyping ("just ship it, bro")

### 3. Easter Eggs & Lyrics
- More Sonu Nigam lyrics mapped to iteration counts
- Iteration-specific GainAm.com messages
- ASCII art for Golden Mic celebrations
- Cultural references from other music traditions

### 4. Translations
The motivation system needs:
- Tamil (Ilayaraja, AR Rahman lyrics)
- Telugu (Tollywood bangers)
- Korean (K-drama coding vibes)
- Spanish (reggaeton debugging energy)
- Japanese (anime determination quotes)

### 5. Anu Malik Response Screenshots
Share your best `/ask-anu-malik` screenshots! We'll feature the most devastating burns in the README.

## How to contribute

1. Fork → branch → make changes → test with `claude plugin install ./your-fork` → PR
2. Tag PRs: `judge`, `roast`, `easter-egg`, `translation`, `skill`, `docs`

## Hall of Fame

Contributors with merged PRs earn **Nigam Crew** status. Top contributors get featured in the README as "Guest Judges."

🎤 "Abhi mujh mein kahin" — The answer is within the community.
